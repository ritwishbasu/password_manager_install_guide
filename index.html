Action Required: Password Manager Setup

Complete these steps to verify compliance for your work computer.

1

Install only one out of these Approved Password Managers

Select only one authorized password manager out of those choices and install its desktop app and browser extension:

1Password
App Extension
Bitwarden
App Extension
LastPass
App Extension
Dashlane
App Extension
2

Keep the Vanta App Active

Ensure the Vanta Device Monitor app is running in the background on your system.

How to verify Vanta is active on your machine:

  • Linux: Run sudo /var/vanta/vanta-cli status in Terminal
  • macOS: Open Activity Monitor and look for Vanta
  • Windows: Open Task Manager and look for Vanta Device Monitor
