Action Required: Password Manager Setup

Complete these steps to verify compliance for your work computer.

1

Install an Approved Password Manager

Select an authorized password manager and install its desktop app or browser extension:

1Password
App Extension
Bitwarden
App Extension
LastPass
App Extension
Dashlane
App Extension
💡 Crucial Step: Installing the browser extension in your main browser (Chrome, Safari, Edge, etc.) enables Vanta to immediately verify compliance.
2

Keep the Vanta App Active

Ensure the Vanta Device Monitor app is running in the background on your system. It continuously scans and confirms your security setup automatically.

✨ Automated Verification: You do not need to send screenshots or manual evidence—Vanta updates in real time once active.
3

Verification Complete

Once Vanta syncs with your system, your machine status will automatically flip to Compliant on the IT security dashboard.
